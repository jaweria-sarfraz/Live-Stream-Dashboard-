# Live-Stream-Dashboard-
This is a dashboard application that successfully integrates C# WinForms with a Python backend and SQL Server. This project demonstrates how a C# interface can trigger and manage data-driven visualizations powered by Python.

## Technologies Used
- **Frontend:** C# WinForms
- **Backend:** Python
- **Database:** Microsoft SQL Server

## Key Features
- **Seamless Integration:** A C# GUI button that triggers the Python-based data dashboard.
- **Data Management:** Integrated SQL Server database for handling application data.
- **Responsive UI:** Minimalist and high-contrast design.

## How to Run
1. Ensure you have Python and SQL Server installed on your system.
2. Clone the repository.
3. Configure the SQL Server connection string in the application settings.
4. Run the C# project to launch the interface and trigger the Python dashboard.
